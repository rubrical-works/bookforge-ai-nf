# Project Charter: AI-Assisted Nonfiction Authoring Framework

**Status:** Active
**Last Updated:** 2026-01-05

## Vision

A production-ready framework for writing professional nonfiction books with AI assistance using Claude Code, providing structured workflows, version control integration, and intelligent prompts while ensuring factual accuracy through anti-hallucination guidelines.

## Current Focus

v0.17.0 - Framework maintenance and Claude Code integration improvements

## Tech Stack

| Layer | Technology |
|-------|------------|
| Language | Markdown, Bash/Batch |
| Framework | Claude Code + IDPF-Agile |
| Integration | Git, Pandoc |

## In Scope (Current)

- 16 conversational prompts for book authoring workflow
- Change tracking system with _chg.md files
- Writing style management (19 curated + custom styles)
- Export formats (DOCX, PDF, EPUB)
- Anti-hallucination guidelines for AI accuracy
